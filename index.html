Tipos de elementos

los elementos reemplazables no se tienen que cerrar

  1. imagenes: se reemplaza automaticamente por un elemento
  2. input es un widget que cambiará su apariencia dependiendo del tipo de entrada

algunas etiquetas tienen multiples atributos

atributos generales

  • class: es un atributo general y puede aplic?arse a varios elementos aplicar estilos y logica
  • id: identifica el elemento y es unico para aplicar logica con js y demas
  • hidden: ocultar elemento si existe el atributo en la etiqueta

atributos especificos

    1. imagenes: alt es importante para mejorar el seo
    2. imagenes: src es la ruta de la imagen
    3. imagenes: title
    • input: type define el tipo de input
